Résumé du Cours CSS

Ce cours aborde les principes de base du CSS pour styliser les pages web.

Qu'est-ce que le CSS ?

CSS (Cascading Style Sheets) est un langage de style utilisé pour définir l'apparence et la mise en page d'un document HTML. CSS permet de modifier des éléments tels que les couleurs, les polices, les marges et les espacements.

La structure de base du CSS

Le CSS se compose de sélecteurs et de déclarations. Une déclaration contient une propriété et une valeur. Voici un exemple de base :

selector {
    property: value;
}

Par exemple :

h1 {
    color: blue;
}

Ce code applique la couleur bleue à tous les éléments `

` de la page.

Les types de sélecteurs

  • Sélecteur de type : Cibler tous les éléments d'un type, par exemple p pour tous les paragraphes.
  • Sélecteur de classe : Cibler les éléments ayant une classe spécifique, par exemple .mon-classe.
  • Sélecteur d'ID : Cibler un élément avec un ID spécifique, par exemple #mon-id.
  • Sélecteur universel : Cibler tous les éléments de la page avec *.

Propriétés CSS courantes

  • color : Définit la couleur du texte (par exemple, color: red;).
  • background-color : Définit la couleur de fond (par exemple, background-color: #f0f0f0;).
  • font-family : Définit la police du texte (par exemple, font-family: Arial, sans-serif;).
  • margin : Définit l'espace autour des éléments (par exemple, margin: 10px;).
  • padding : Définit l'espace à l'intérieur des éléments (par exemple, padding: 15px;).

Les Box Models en CSS

Le modèle de boîte (box model) définit la façon dont les éléments sont affichés dans une page web. Un élément peut être constitué de plusieurs parties :

  • Content : Le contenu de l'élément (texte, image, etc.).
  • Padding : L'espace autour du contenu, à l'intérieur de la bordure.
  • Border : Une bordure autour du padding.
  • Margin : L'espace à l'extérieur de la bordure.

Voici un exemple de CSS qui modifie le box model :

div {
    width: 200px;
    padding: 20px;
    border: 1px solid #000;
    margin: 10px;
}

Conclusion

Le CSS est essentiel pour styliser et disposer les éléments HTML sur une page web. En comprenant les sélecteurs, les propriétés et le modèle de boîte, vous pouvez créer des designs web attrayants et fonctionnels.
