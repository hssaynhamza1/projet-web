Résumé du Cours Langage C

Ce cours couvre les bases du langage de programmation C, utilisé pour développer des logiciels et des systèmes embarqués.

Introduction au Langage C

Le langage C est un langage de programmation impératif créé dans les années 1970. Il est largement utilisé dans le développement de logiciels système, d'applications embarquées et dans la programmation de bas niveau.

Structure d'un Programme C

Un programme C de base se compose généralement de la déclaration des bibliothèques, de la fonction main() et des instructions. Voici un exemple simple de programme C :

#include 

int main() {
    printf("Bonjour, le monde !\n");
    return 0;
}

Dans cet exemple, #include est utilisé pour inclure la bibliothèque standard d'entrée/sortie qui permet d'afficher des messages à l'écran. La fonction main() est le point d'entrée du programme.

Les Variables et Types de Données

En C, les variables sont utilisées pour stocker des données. Chaque variable doit être déclarée avec un type spécifique. Voici quelques types de données courants en C :

  • int : Pour les entiers.
  • float : Pour les nombres à virgule flottante.
  • char : Pour les caractères (ex. : 'a', 'b', etc.).
  • double : Pour les nombres à virgule flottante avec plus de précision.

Exemple de déclaration de variables :

int age = 25;
float poids = 70.5;
char sexe = 'M';

Les Structures de Contrôle

Les structures de contrôle permettent de définir le flux d'exécution d'un programme. Voici les principales structures de contrôle en C :

  • Conditionnelle (if, else) : Permet d'exécuter des blocs de code en fonction de conditions.
  • Boucles (for, while, do-while) : Permettent de répéter des instructions plusieurs fois.

Exemple d'une condition if :

if (age > 18) {
    printf("Vous êtes majeur.\n");
} else {
    printf("Vous êtes mineur.\n");
}

Les Fonctions en C

Les fonctions sont utilisées pour organiser le code en blocs réutilisables. Une fonction en C peut être définie comme suit :

type_de_retour nom_de_fonction(paramètres) {
    // Corps de la fonction
    return valeur;
}

Exemple d'une fonction qui additionne deux nombres :

int additionner(int a, int b) {
    return a + b;
}

La fonction additionner prend deux entiers en entrée et retourne leur somme.

Pointeurs et Mémoire en C

Les pointeurs en C sont des variables qui stockent l'adresse mémoire d'une autre variable. Cela permet une gestion plus fine de la mémoire.

Exemple de déclaration et d'utilisation d'un pointeur :

int x = 10;
int *ptr = &x;  // pointeur vers x
printf("%d\n", *ptr);  // affiche la valeur de x

Ici, ptr est un pointeur qui pointe vers la variable x, et en utilisant *ptr, nous accédons à la valeur stockée à l'adresse mémoire de x.

Conclusion

Le langage C est un outil puissant qui offre un contrôle total sur la gestion de la mémoire et l'exécution des programmes. En maîtrisant les bases comme les variables, les structures de contrôle, les fonctions et les pointeurs, vous pouvez développer des applications performantes et optimisées.
