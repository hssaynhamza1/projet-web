Résumé du Cours JavaScript

Ce cours couvre les bases de JavaScript, le langage utilisé pour rendre les pages web interactives.

Qu'est-ce que JavaScript ?

JavaScript est un langage de programmation principalement utilisé pour créer des effets dynamiques et interactifs sur les pages web. Il permet de manipuler les éléments HTML et CSS d'une page web en temps réel.

La Structure de Base d'un Script JavaScript

Un script JavaScript peut être ajouté à une page web en utilisant la balise <script>. Voici un exemple de structure de base d'un script JavaScript :

<script>
    // Ceci est un commentaire
    alert("Bonjour, le monde!");
</script>

Dans cet exemple, un message pop-up apparaît sur la page lorsque le script est exécuté.

Les Variables en JavaScript

Les variables en JavaScript sont utilisées pour stocker des valeurs. On utilise les mots-clés var, let, ou const pour déclarer une variable.

  • var : Ancien mot-clé pour déclarer des variables (moins utilisé aujourd'hui).
  • let : Déclare une variable dont la valeur peut être modifiée.
  • const : Déclare une variable dont la valeur ne peut pas être modifiée.

Exemple :

let x = 10;
const y = 5;

Les Fonctions en JavaScript

Une fonction est un bloc de code que l'on peut exécuter plusieurs fois. Voici un exemple de fonction simple :

function saluer() {
    alert("Bonjour!");
}

Pour appeler la fonction et exécuter son code, on écrit :

saluer();

Les Conditions en JavaScript

Les structures conditionnelles permettent d'exécuter un code en fonction de certaines conditions. Par exemple, la structure if permet de vérifier une condition :

if (x > y) {
    alert("x est plus grand que y");
} else {
    alert("x n'est pas plus grand que y");
}

Les Boucles en JavaScript

Les boucles sont utilisées pour exécuter un bloc de code plusieurs fois. Voici un exemple d'une boucle for :

for (let i = 0; i < 5; i++) {
    console.log(i);
}

Dans cet exemple, le code affichera les nombres de 0 à 4 dans la console.

Les Événements en JavaScript

JavaScript peut être utilisé pour gérer les événements sur une page web, comme un clic sur un bouton ou un mouvement de souris. Exemple :

document.getElementById("monButton").onclick = function() {
    alert("Le bouton a été cliqué!");
};

Ce code attend que l'utilisateur clique sur un bouton avec l'ID "monButton" et affiche un message lorsque cela se produit.

Conclusion

JavaScript est un langage puissant qui permet de rendre les pages web interactives et dynamiques. En comprenant les variables, les fonctions, les conditions, les boucles et les événements, vous pouvez commencer à créer des pages web plus complexes et interactives.
